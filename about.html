How can I contribute?

To contribute to Lenni's original branch, you can fork the repo, edit the code, and then open a pull request back to the main repository when you're done.

To contribute to this branch, you can fork the repo, edit the code, and then open a pull request back to this repository when you're done.

If you come across a bug and want to report it, use either the feedback form, message me on Reddit or Discord, or submit a GitHub issue.

Can I make my own version?

Yes, absolutely!

The app is licensed under the GPLv3 license, meaning you can make your own version, as long as it stays open source and also has the GPLv3 license.

Do you collect data?

No, the app collects no data. The only data that is saved to your local drive is your currently selected theme. Everything you do here stays on your local computer.
